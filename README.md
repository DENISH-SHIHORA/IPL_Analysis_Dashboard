# 🏏 IPL Analytics Dashboard (2008–2025)

## 📌 Project Title

**IPL Analytics Dashboard – Power BI**

---

## 📝 Short Description

An interactive Power BI dashboard designed to analyze IPL seasons (2008–2025), providing insights into team performance, player statistics, and tournament trends through dynamic visualizations and KPIs.

---

## 🛠️ Tech Stack

* **Power BI** – Data visualization & dashboarding
* **DAX (Data Analysis Expressions)** – Calculated measures & KPIs
* **Excel / CSV** – Data preprocessing and storage

---

## 📂 Data Source

* Historical IPL datasets (matches, players, teams)
* Structured data including:

  * Match results
  * Player statistics
  * Team standings

---

## ⭐ Features / Highlights

* Season-wise filtering (dynamic analysis)
* Champion & Runner-up summary
* Orange Cap & Purple Cap insights
* Batting & bowling performance tracking
* Boundary analysis (4s & 6s)
* Interactive Points Table
* KPI cards for quick metrics overview
* Clean and user-friendly dashboard layout

---

## ❓ Key Questions Answered

* Which team performed best in a given season?
* Who were the top run scorers and wicket takers?
* How competitive was the league (points distribution)?
* Which players dominated in boundaries (4s & 6s)?
* What are the overall scoring trends in a season?
